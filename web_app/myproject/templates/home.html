Welcome to Financinos

The purpose of this web application is for downloading financial statement data and automating valuation models which estimate the intrinsic value of a stock price. This database contains cleaned data from SEC's EDGAR. Using this database, there are two ways in which you can download data:

  • Download data via CSV or API

Two different valuation techniques used to find a company's true stock price value in this application and a compounding interest calculator:

  1. Discounted Cash Flow Model

  2. Price-Earnings Multiple Model

  3. Compounding Interest Calculator - On top of using the two models to find value stocks. Use the calculator to discover the earnings power of holding on to stocks long term

Providing access to financial data on thousands of United States companies. With quarterly and annual data available for downloading and analysis, this app is completely free to use
